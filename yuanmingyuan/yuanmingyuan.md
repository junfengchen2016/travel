## 圆明园
- 万花阵
```
陛下,现在我们来到的这个地方可是圆明园的迷宫哦,这里是重新复建的万花阵。万花阵花园门为西洋座钟形,黄铜雕刻花纹大门两扇。花园迷宫的中央为石亭,内设西洋式座椅,是仿照欧洲的迷宫而建的花园。它的主要特点是:用四尺高的字图案的雕花砖墙,分隔成若干道迷阵,因而称作"万花阵"。盛时,每当中秋之夜,清帝坐在阵中心的圆亭里,宫女们手持黄色彩绸扎成的莲花灯,寻径飞跑,先到者便可领到皇帝的赏物。所以也叫黄花阵或黄花灯。虽然从入口到中心亭的直径距离不过30多米,但因为此阵易进难出,容易走入死胡同,清帝坐在高处,四望莲花灯东流西奔,引为乐事
```
- 大宫门
```
此处的大宫门为绮春园宫门,建成于嘉庆十四年(1809年),因它比圆明园大宫门和长春园二宫门晚建半个多世纪,也称"新宫门”,一直沿用至今。陛下,新宫门坐北朝南,南起宫门广场前大影壁,北至中和堂前的寿山,南北长近200米,占地面积24万平方米。新宫门前建东西朝房各五间,正南有长近30米的大影壁,在影壁与宫门之间用木栅栏围成一个小广场。影壁外侧为环园护城河,在护城河两侧还建有皇太后外膳房、奶茶院等大小建筑五十五间。宫门为五间卷棚歇山顶门殿,门内檐挂乾隆御书“绮舂园″金字匾,宫门为皇帝与皇太后出入专用。在宫门的两侧还分别建有两个角门,王公大臣奉旨入园需从东面的角门进园,而太监、杂人则由西面的角门进出园子。大宫门后原有一道月牙河,河中有一座汉白玉石桥,过石桥是五间穿堂门殿,规格略小于迎晖门,俗称二宫门”。二宫门的两边还各建有座角门,与大宫门一样皆为王公大臣及太监、杂人出入
```
- 天然图画
```
天然图画为圆明园四十景之一,坐落在九州清晏景区的东岸,占地面积一万六干平方米。景区内种植大量的竹子,以翠竹著名,雍正为皇四子时赐名“竹子院”,也是嘉庆皇帝小时居住的地方。这座建筑西高东低,西南面有著名的朗吟阁,现故宫博物院内有一幅人物肖像画,便是雍正在朗吟阁的画像。在朗吟阁的南边还有·座建筑,叫“五福五代堂”,并由乾隆御书,因为乾隆七十四岁时得到玄孙而赐名,康熙帝名字叫“玄烨",所以清代不能随便用“玄"字,乾隆帝便用″元″代替“玄"字。在景山的景福宫、承德避暑山庄、天然图画都有乾隆御书“五福五代堂匾,在此处的东边有一桥廊,便是仿自西湖十景之一的"苏堤春晓″,也是由乾隆御书。陛下请看,这座建筑的南部池中有小岛,以自然布列的汀步与岸上相联系。小岛上叠石培土,植有几株青松。池的北岸,靠漏窗花墙处,重点点缀了几棵花木。池的南岸有栽满乔木的土山作为绿色的屏风。南北取材不同,整个园林的色彩既丰富又能随着时令而变化。夏天,竹喧荷静,清香四溢,生机勃发,使人“绿满襟袖″。严寒季节,也并非一片光秃,自有长绿的青松和翠竹与白雪相映,组成一幅冬令时节的天然图画。
```
- 平湖秋月
```
平湖秋月为圆明园四十景之一,位于福海北岸,建自雍正朝,境仿杭州西湖同名景,为秋夜赏月佳处,由一组散布的临水建筑组成。每逢清风徐徐的秋夜,这里湖平如镜、月光如昼、桂花飘香,典雅的园林建筑与湖面、花木、皓月、倒影,融洽协调的组成娴静优美的图画,真是“万顷湖平长似镜,四时月好最宜秋”,令人遐想联翩,回味无穷。无怪乎乾隆要说,这里比杭州西湖的苏公堤还要更加吸引人。
```
- 正觉寺
```
正觉寺位于绮春园正门西,占地面积1266万平方米,于乾隆三十八年(1773)就已建成,是绮春园中最大的佛寺,俗称“喇嘛庙"。正觉寺建成后乾隆皇帝从香山宝谛寺调来喇嘛四十一人,其中首领喇嘛一人,小喇嘛四十人。每月初一、初八、十三日、十五日这些喇嘛都要在正觉寺内念经。乾隆皇帝也不定期地前来正觉寺颂经。正觉寺坐北朝南,有独立的山门通向园外,同时又与后门及绮春园内相通,是圆明三园唯一一组独立通向园外的景区。山门为三开间,门外檐石刻乾隆御书汉、满、藏、蒙四种文字的“正觉寺”匾,正觉寺山门东、西建有旁门一座,此山门只有在正觉寺举行重大活动时才开启,平时僧侣只能从旁门进出。山门殿塑有哼哈二将,形象威武凶恶。进入山门后是天王殿五间,殿中供奉有大肚弥勒佛一尊,殿内东、西两侧是四大天王塑像。天王殿前东、西分别建有钟、鼓楼,楼前各立有旗杆一根。天王殿后为正殿七间,名叫“三圣殿”。殿前、后有廊,殿后有抱厦三间。殿前有月台,殿内正中塑楠木三世佛雕像,东、西分别列十八罗汉彩塑。后抱厦內塑南海大士一尊。天王殿与三圣殿东、西分别建有配殿五间,均为喇嘛住房。
```
- 武陵春色
```
这是仿照晋代陶渊明《桃花源记》中所描绘的意境建造的。陛下,它位于万方安和背面,占地面积2000平方米,为圆明园四十景之一。武陵春色在康熙年间就已建成,初称"桃花坞”,乾隆皇帝小时候就曾被赐居在这里。它修建得十分有特点,分东西两部分,建筑全集中在西部,东部则是一条水溪。水溪两岸种植了大量桃花,所以溪又叫“桃花溪”,如果人想进入西部景点,需要乘船前往逆流而上,穿过一座用青石架成的桃花洞,才可进入“世外桃园"。“世外桃园"由三组建筑组成,最南面是一个封闭的院落,院內主体建筑叫“全碧堂”,是专为帝后演戏的太监居住的地方。中间建有一组戏台分观戏殿与戏台两组建筑。观戏殿外檐下悬挂有皇帝御笔“恒舂堂”匾,南面则是一座两层戏台。嘉庆、道光年间两位皇帝经常在这里陪太后看戏。戏台北面是由几个大小不同的建筑组成,是乾隆皇帝做皇子时读书的地方。后来乾隆老年时还经常来这里读书,回忆自己的童年时光。
```
- 澄心堂
```
澄心堂位于绮春园西路南湖岛上,占地面积1.5万平方米。它在乾隆时期就已归入绮春园,最初这里叫“竹园″。嘉庆中后期进行过大规模修缮。澄心堂的正殿就坐落在岛上偏南的位置,坐北朝南,为两卷五间大殿。殿内装修十分豪华,里面设有宝座床。咸丰皇帝曾多次在此陪同皇太妃用膳。澄心堂东西各建有七间套殿,俗称“西室”、“东室"。西室外檐下悬挂“垂虹榭”匾,东室外檐下悬挂“绮旭车”匾。澄心堂的北边建有东、西配殿各五间,是库房与值房。嘉庆、道光及咸丰祖孙三代皇帝对澄心堂都很喜欢,也经常在澄心堂用膳、休息。澄心堂东北小岛上是三开间敞厅湛清轩"。岛的北岸与其隔湖相望建有坐北朝南五间两卷殿,名叫“云漪馆",而澄心堂西南对岸则是“凌虚亭”,外檐悬嘉庆御书“凌虚亭”匾
```
- 狮子林
```
狮子林位于长舂园东湖北岸,建造于乾隆年间,是仿苏州狮子林建造的。乾隆皇帝下江南时,曾经在苏州游览过著名的园林狮子林,非常喜爱这座园林,于是便要求在圆明园内仿建这座精致的园林,圆明园中有四十余处仿制江南园林的景点。乾隆皇帝曾经十次题咏“狮子林十六景”,匾诗刻石非常多。1994年公园管理处局部清整了狮子林遗址临水的基础和河道,发掘恢复虹桥、水关及水门三座石拱,并复位保护乾隆御笔匾、诗刻石32件。
```
- 福海
```
福海是全园最大的水面,也是皇家御园的水上游乐中其实,它在雍正朝命名之前俗称东池或东湖,开凿年代可能为康熙末叶。经雍正即位后的进一步开拓才有后来的规模。陛下是不是很疑惑,这明明是湖却怎么会被命名为"海”呢,聪明的陛下肯定也猜到这是有原因的。相传,东海中有三座神山,山上有仙人居住,还有长生不老药。秦始皇曾派徐福率童男童女数干人,入海寻仙境、求仙药,福海的命名正是取"徐福海中求″的寓意,以求皇帝长生不老,大清帝国江山永固。福海位于圆明三园的中心地带,以辽阔开朗取胜。水面近于方形,宽度约五、六百米,总面积约为28万平方米,加上周围的小水域共计32万平方米。历史上福海最深处达3.8米,一般水面深度也有23米左右。这样的深度为每年端午龙舟竞渡,及平日大船行驶提供了保证。在长达一百三、四十年的时间里福海给皇家的御园生活增添了无穷的乐趣,但是它也并没有给皇家带来幻想中的福气,相反倒成了埋葬圆明园管园大臣的水中墓穴1860年夏,咸丰皇帝在福海最后一次享受了泛舟之乐,此后不久,英法联军攻打北京,咸丰皇帝仓皇逃往承德。10月,圆明园被英法联军劫掠、焚毁。管园大臣文丰无力组织保卫,跳入福海自尽。好在1984年12月1日,圆明园遗址公园开始全面整修,福海景区作为第期整修工程于1985年6月29日竣工开放。经多年整修,如今的福海景区已是一处难得的山水园林佳境,成为圆明园遗址公园的主要景区之一。陛下在这里虽然看不到宏伟壮观的景色,但盛时的格局仍大体存在;山形水系已基本恢复了历史原貌;从浩瀚的湖面与环绕的河溪中依稀可见山水相依、烟水迷离的景致。身临其境,仿佛可以回到昔日美妙的园林图画。而″吴宫花草埋幽径"的格调,更符合遗址园林的气质,给人一种沧桑感和关于历史的强烈震撼。
```
- 绮春园
```
圆明园是由圆明园、绮春园和长春园三个园子组成的。绮春园原是怡亲王允祥的赐邸,约于康熙末年始建,后曾改赐大学士傅恒,至乾隆三十五年(1770年)正式归入御园,定名绮春园。嘉庆四年和十六年,该园的西部又先后并进来两处赐园,一是成亲王永瑆的西爽村,是庄敬和硕公主的含晖园,经大规模修缮和改建、增建之后,该园始具干亩规模,成为清帝园居的主要园林之。至此,圆明三园处于全盛时期,全园占地16万平方米,有景点百余处。被毁前的绮春园面积比长春园略小,为54.3公顷。由于绮春园是由若干个小园子合并而成,所以全园没有一个统一、完整的布局。各园之间相对独立,但园林设计者又将河道彼此相连通,使各个独立的小园子又相互串联起来,使之联成一个整体,这也是绮春园不同于圆明园与长春园之处。
```
- 蓬岛瑶台
```
福海之中的蓬岛瑶台,取材于神话中的蓬莱仙岛,原名蓬莱洲。相传,秦始皇曾派遣一个名叫徐福的人,率领若干名童男童女,出海东渡,替他寻仙境、求仙药,以祈求长生不老。这当然只能是"海客谈瀛洲,烟涛微茫信难求”。而雍正皇帝则让工匠在圆明园的东湖之中用磷峋巨石堆砌成大小三岛,象征传说中的篷莱、瀛洲、方丈"三仙山",岛上建有殿阁享台,"望之若金堂五所,玉楼十二",并按"徐福海中求"的寓意,把东湖命名为"福海"。在福海四岸另外还建有十多处园林佳景。福海,东西、南北各宽五、六百米,加上四周小水域,共约35公顷,相当于北海公园的水面。这里水面开阔,景色秀丽,每当端午佳节,在此举行大型龙舟竞渡活动。七月十五日夜,皇帝于此观赏河灯。冬日结冰后,皇帝乘坐冰床在福海赏游。福海实际上是圆明园的水上娱乐中心。
```
- 西洋楼
```
明清时期,西方文化传入中国,从建筑和科技领域开始融入中国。当时的皇家园林圆明园就引进了欧式风格,建造了“西洋楼”,由谐奇趣、线法桥、万花阵、养雀笼、方外观、海晏堂、远瀛观、大水法、观水法、线法山和线法墙等十余个建筑和庭园组成,建筑材料多用汉白玉石,石面精雕细刻,屋顶覆琉璃瓦,占地约7公顷。西洋楼于乾隆十二年(1747年)开始筹划,至二十四年(1759年)基本建成,由西方传教土郎世宁、蒋友仁王致诚等设计指导,中国匠师建造。建筑形式是欧洲文艺复兴后期"巴洛克"风格,造园形式为"勒诺特"风格但在造园和建筑装饰方面也吸取了不少我国传统手法。西洋楼的主体,其实就是人工喷泉,时称"水法"。特点是数量多、气势大、构思奇恃。主要形成谐奇趣、海晏堂和大水法三处大型喷泉群,颇具殊趣。1860年“火烧圆明园”时被焚毁。1977年至1992年间,西洋楼各遗址得以清理,廓清殿座基址,整修喷泉池,归位柱壁石件,并修复了迷宫“黄花阵"。
```
- 迎晖殿
```
这是绮春园的正殿,位于二宫门内,嘉庆时称“勤政殿”,道光皇帝即位后更名为“迎晖殿”,并亲笔御书匾额挂在大殿外檐下。迎晖殿为五开间,四周设有围廊,殿前有月台,殿内有宝座。道光时期每到皇太后万寿节,皇太后便由畅春园搬至绮春园,在此殿内接受群臣、众妃们朝贺。道光二十九年(1849)孝和睿皇太后病逝,咸丰五年(1855)康慈皇贵太妃病逝,都先后在迎晖殿内停灵祭奠,之后才发丧至陵寝。
```
- 鉴碧亭
```
陛下,咱们眼前的这座亭子名叫鉴碧亭,它原建于1811年,即嘉庆十六年前后,位于绮春园宫门内,碧湖之中,方亭高敞,为观景佳处。1993年在原址修复重檐方亭,建筑面积256平方米。鉴碧亭是圆明园遗址公园的重要景观。
```
- 长春园
```
长春园作为圆明园三园之一,始建于乾隆十年1745年前后,于1751年正式设置官园总领时,园中路和西路各主要景群已基本建成,诸如詹怀堂、含经堂、玉玲珑馆、思永斋、海岳开襟、得全阁、流香渚、法慧寺、宝相寺、爱山楼、转湘帆、丛芳榭等。其后又相继建成茜园和小有天园。而位于长春园东部的映清斋、如园、鉴园、狮子林,是乾隆31年至37年大规模增建的,包括西洋楼景区,长春园共占地一千亩。悬挂匾额的园林建筑约有200多座。
```
